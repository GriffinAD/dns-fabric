# Internationalization Architecture

## Scope

Defines message-format strategy, locale negotiation, plugin-contributed strings,
and validation for multilingual readiness.

## Acceptance criteria for Rolling close

- [ ] Locale strategy and fallback chain documented.
- [ ] Extraction/validation workflow defined in CI.
- [ ] Plugin string contribution boundary documented.
- [ ] RTL and pseudo-locale testing criteria specified.

## I18n outline

Use ICU-style message formatting with deterministic key namespaces and
build-time validation for completeness.
